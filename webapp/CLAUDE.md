## Project Overview

**ETT (English Talk Time)** is a Brazilian English language acceleration program targeting tech professionals (Data, AI, BI, Cloud, Technology). The site is a **greenfield project** — there is no code yet. The `ConteudoSite/` folder contains planning documents, brand assets, and partner logos used to build the site.

The project is written/documented in **Portuguese (pt-BR)**.

## Program Architecture (from `ProjetoGeralETT.docx`)

The ETT site must support a **6-pillar learning ecosystem**:

1. **Commitment & Community** — Practice community, events, challenges, rankings, progress tracking
2. **Structured Learning Base** — Onboarding, leveling assessment, integration with BeeTools partner courses
3. **Real Practice & Guided Conversation** — English Talk Time sessions (simulations, role-play, storytelling)
4. **Intensive Immersion** — Partnership with Cherry Top immersion programs
5. **International Employability** — Career mentoring via Coders partner (CV, LinkedIn, interviews)
6. **Intelligence & Personalization** — AI-driven vocabulary gap analysis and study recommendations

### Key Tools to Build (section 9 of `ProjetoGeralETT.docx`)

| Tool | Purpose |
|------|---------|
| Vocabulary Diagnostic + Study Plan | Identifies gaps, generates personalized plan targeting 3,000 words |
| "Technique of Dots" / Smart Revision | Daily new words + selective review of weak words (10-20 words/day) |
| Series & Subtitles Tool | SRT-based study with unknown word marking and listening practice |
| Reading + Audiobook Module | Guided reading with audio; integrates Lit2Go public domain library |
| Audio Hub | Radio, news, talk shows for advanced listening immersion |
| Written/Spoken/Feelings Diary | Personal expression tool to activate passive vocabulary |
| Personalized Printed Materials | QR-code-linked PDFs, word lists, study plans |
| Intelligent Recommendation System | Uses click/error data to generate flashcards and prioritized exercises |
| AI Prompt & Simulation Tool | Generates custom prompts for ChatGPT/AI to simulate real situations (immigration, interviews, meetings) |

### Partners

- **BeeTools** — Structured base course (levels A0-A2)
- **Cherry Top** — Immersion programs
- **Coders** — International career mentoring
- **IEP** — (English instruction partner)

### Target Audience

Tech professionals and students (Data, AI, BI, Cloud) who already have some English exposure but are blocked from speaking. Program targets ~300 hours of contact and ~3,000 vocabulary words to reach functional international English.

## Business Model Notes

- At least one free monthly event (up to 100 people) as the entry funnel
- Free community access before paid program tiers
- Paid acceleration layers for immersion, tools, and career support
- Intelligent system data can be shared (with consent) with teachers/partners for personalized coaching
